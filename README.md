# 🤖 Advanced Rule-Based Chatbot (Python)
---
An advanced rule-based chatbot built using Python.  
This project demonstrates intent matching, dynamic responses (date/time), and multi-intent detection.

But to make more chat and conversation , you need to add pattern & responsed based on your convenient and application.

## 🚀 Features

- Pattern-based intent recognition using Regular Expressions
- Multi-intent handling (e.g., "what is the date and time?")
- Dynamic date and time responses
- Modular project structure
- Beginner-friendly and extendable

---

## 📂 Project Structure

```
advanced-rule-based-chatbot/
│
├── chatbot.py       # Main chatbot engine
├── utils.py         # Intent matching & response logic functions
├── intents.py       # Chatbot knowledge base (add more data here)
└── README.md
```

---

## 🛠 Technologies Used

- Python 3
- Regular Expressions (`re`)
- Datetime module

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/advanced-rule-based-chatbot.git
```

2. Navigate into the folder:

```
cd advanced-rule-based-chatbot
```

3. Run the chatbot:

```
python chatbot.py
```
